# linux basic command

- `pwd` ( print working directory)
    - 현재 작업중인 경로를 표시)
- `clear` (`crtl + l`)
    -터미널 창을 지워주는 명령어
- `ls`
    - 현재 폴더에 있는 파일/폴더 목록 표시
    - `-a` (all) (option) : 숨겨진 목록까지 표시
- `mkdir {directory_name}`s
    -폴더 생성
- `touch {file_name}`
    - 파일 생성
- `rm {name}`
    - 파일 삭제
    - `-r`(recursive) (option) : 폴더(재귀) 삭제

- `cd` (change directory)
    - `..` 위로가기
    - `cd {directory_name}` : 해당 폴더로 이동
    - ~ : 최상위 폴더
    
    - .. : 상위
    - . : 현위치
